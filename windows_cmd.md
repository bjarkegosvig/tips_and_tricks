# Set PYTHONPATH for current session
**To replace PYTHONPATH with wanted path use:**  
```set PYTHONPATH=C:\My_python_lib```

**To append to PYTHONPATH use:**  
```set PYTHONPATH=%PYTHONPATH%;C:\My_python_lib```
